# Conecta4-Java
Primer Reto Mensual de Mouredev

## Reglas 
- El conecta 4 se juega en una rejilla vertical de 6 filas y 7 columnas.
- El objetivo es alinear 4 fichas de su color, horizontalmente, verticalmente o en diagonal.
